# ChatApp

Приложение для чата.

## Стэк технологий

- Laravel
	- Jetstream
	- Sanctum
- Vue

## Как запустить

Для запуска понадобится сервер с PHP и MySQL. Если вы на Windows, то подойдет Xampp. Так же нужно установить Composer (для Windows лучше использовать установщик, т.к. он позволит использовать composer глобально).

- Склонируйте проект в папку `xampp/htdocs/`
- Запустите shell из панели управления Xampp
- В shell перейдите в папку проекта и выполните команду `composer install`. Это установит Laravel со всеми зависимостями.
- Далее нужно запустить `npm install`, чтобы установить Vue
- Создайте базу данных
- Отредактируйте файл `.env.example`, добавив информацию о базе данных и переименуйте его в `.env`
- Запустите команду `npm run watch`
- Откройте в браузере `localhost/[имя папки с проектом]/public`, должна загрузиться страница с надписью Laravel
- В правом верхнем углу нажмите Register, чтобы зарегистрировать нового пользователя
